# CMPM121-F25-FINAL

## How we satisfied the software requirements

We are using the Love2D framework and its corresponding Lua from Professor Adam's lecture notes. This platform meets the requirement that it "does not inherently provide support for 3D rendering and physics simulation." The third-party 3D rendering library is from Luke100000, and the third-party physics simulation library is from HDictus. We simply need to put them in the lib file and reference them in main using local = require(""). The simple physics-based puzzle involves manipulating one block to push another block to a designated (green) area. The player must manipulate the block, otherwise, they will fail after a certain time. Success or failure is indicated by text in the top left corner. Currently, we have tried using before-commit automation to check code uploads, but it is not yet complete.
